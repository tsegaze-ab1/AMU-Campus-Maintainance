Frontend Export Package (PHP Integration Ready)

This folder contains a standalone frontend bundle extracted from your designed homepage.

Included:
- index.html
- css/bootstrap.min.css
- css/style.css
- js/main.js
- lib/ (animate, easing, lightbox, owlcarousel, waypoints, wow)
- images/ used by the homepage
- images/dashboards/ used by the homepage

Notes:
- This export was created without modifying the original project files.
- External CDNs are still used in index.html for fonts/icons/jquery/bootstrap js.
- You can drop this folder inside your PHP project and start wiring routes/forms.
